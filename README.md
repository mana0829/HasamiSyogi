# はさみ将棋 Webアプリ

スマートフォン（特にAndroid）で動作する、シンプルな「はさみ将棋」Webアプリです。  
ブラウザ（Chrome）でそのまま遊べます。

---

## 🎮 ゲーム概要

はさみ将棋は、駒で相手の駒を「挟んで取る」シンプルなボードゲームです。

- 盤面：5×5
- プレイヤー：黒 vs 白
- ターン制

---

## 🧩 ルール

### ▼ 基本ルール

- 駒は縦・横に何マスでも移動可能
- 他の駒を飛び越えることはできない
- 相手の駒を自分の駒で挟むと取れる
- 挟みは縦横のみ（斜め不可）
- 複数の駒を同時に取れる

---

## 🏁 勝利条件

### ▼ 通常ルール

- 相手の駒が **0個** になったら勝ち
- 相手が **動けなくなったら** 勝ち

### ▼ 追加ルール（切替可能）

- 「1個になったら負け」チェックボックスあり

| 状態 | 結果 |
|------|------|
| ON | 駒が1個になったら負け |
| OFF | 通常ルールで続行 |

---

## 📱 操作方法

- 駒をタップ → 選択
- 移動可能マスがハイライト表示
- 移動先をタップ → 移動
- 挟んだ駒は自動で削除

## 🧠 実装ポイント

- 盤面は2次元配列で管理
- 移動可能マスを計算
- 挟み判定（複数対応）
- 駒数カウント
- 動けるか判定
- 勝敗判定（ルール切替対応）

---

## 🔧 主な関数

- `init()`：初期化
- `draw()`：盤面描画
- `getMoves()`：移動可能マス取得
- `move()`：駒移動
- `capture()`：駒を取る処理
- `countPieces()`：駒数カウント
- `hasAnyMove()`：動けるか判定
- `checkWinner()`：勝敗判定
